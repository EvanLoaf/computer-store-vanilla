# Computer Store Vanilla

Computer Store Vanilla is a mock store project.

## Requirements
* Java 8
* MySQL Server 8
* Apache Tomcat 9

## Usage

This project showcases the use of:
* Java 8 Core
* Servlets
* MySQL, JDBC
* JSP, JSTL/EL
* JAXB
* XML, XSD
* Regex
* Design Patterns: MVC, Command, etc.
* Git
* Tomcat
* and other development tools
